# Numerical practicals for FDSE 2026

You will do three numerical practicals together

 - #1 **Waves Patterns**: Propagation and dispersion of waves
 - #2 **Basics of RSW**: basic processes with the Rotating Shallow Water model
 - #3 **Buoyancy driven flows**: gravity current and horizontal convection

The documents of the practicals can be found
[here](https://ubocloud.univ-brest.fr/s/ia8k54gLYrAdwtR)

The practicals are based on home-made Python codes: `wave2d` for practical #1
and `Fluids2d` for practicals #2 and #3. They generate interactive animation
during the computation and store the results in the `history.nc` in NetCDF
format. To quickly browse the NetCDF we will use `ncview`.

Before starting we need to do a little bit of installation.


## Install `wave2d`
For that we create a `fdse` virtual environment

```console
cd ~
python3 -m venv ~/fdse
source ~/fdse/bin/activate
git clone https://github.com/pvthinker/wave2d
cd wave2d
pip install -r requirements.txt
```
Note that, you are within the virtual environement, when you see '(fdse)' in the prompt of your terminal. If you are not (e.g. you open a new terminal) and you want to work with this virtual environement, simply do

```console
source ~/fdse/bin/activate
```


## Install `ncview`


```console
mamba init
```

then open a new terminal to make conda active. The prompt line should start with (base)

```console
mamba create -n ncview
mamba activate ncview
mamba install -c conda-forge ncview
```

you now have `ncview`. For a later session, don't forget to do `mamba activate
ncview' to have it. The prompt should start with (ncview)



## Install `Fluids2d`

We use the same `fdse` virtual environment as for `wave2d`

```console
cd ~
source ~/fdse/bin/activate
git clone https://github.com/pvthinker/Fluids2d
cd Fluids2d
pip install -e .

```

Once it is installed and whenever you want to use in a new terminal you need to
activate the `fdse` mamba environment

```console
mamba activate fdse
source ~/python/bin/activate
```

and to test that the code works

```console
cd Fluids2d/src/experiments
python vortex.py
```

should execute your first experiment.

## Add the virtual env to jupyter

```console
cd ~
source ~/fdse/bin/activate
python -m pip install ipykernel
python -m ipykernel install --user --name fdse
```
et voilà !
